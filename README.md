# 🧮 Calculadora no Terminal (Python)

Projeto simples de calculadora desenvolvido em Python para rodar no terminal.

## 📌 Sobre o projeto
Este projeto foi criado com o objetivo de praticar lógica de programação, entrada de dados e controle de fluxo utilizando Python.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Parte inteira de uma divisão
- Resto de uma divisão
- Potenciação
- Execução contínua até o usuário encerrar

## 🛠 Tecnologias utilizadas
- Python

## ▶️ Como executar

1. Clone o repositório:
git clone https://github.com/SanttsFellype/calculadora-terminal-python.git

2. Acesse a pasta:
cd calculadora-terminal-python

3. Execute o arquivo:
python calculadora.py

## 📷 Exemplo de uso
Digite o primeiro número: 10  
Digite o segundo número: 5  
Digite a operação (+, -, *, /): +  

Resultado: 15
